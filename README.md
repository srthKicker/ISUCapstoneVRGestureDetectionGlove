# Hand Gesture Device (ESP32)

## Requirements
- Docker
- VS Code
- ESP32

## Quick Start
1. Clone repo
2. Open in VS Code
3. Make sure Dev Containers and Docker are installed (VSCode Extensions)  // Note Docker Desktop must be open? 
4. "Reopen in Container" Ctrl + Shift + P -> Enter

Inside Terminal of VSCode: 
5. cd DataCollection
6. idf.py build
7. idf.py flash monitor

## Hardware
- ESP32
- 1x MPU9250
- 1x MPU6050
